카드 리스트는 시스템 레이아웃의 그리드 시스템을 사용합니다. 모바일에서는 4컬럼 그리드에서 4컬럼 차지(1개), 태블릿에서는 8컬럼 그리드에서 4컬럼 차지(2개), PC에서는 12컬럼 그리드에서 4컬럼 차지(3개)로 배치되며, 모든 카드는 동일한 크기를 유지합니다. 카드 리스트의 모든 카드는 기본 카드와 동일한 구조를 가지며, 버튼 영역은 제외되어 정보 표시에 집중된 형태로 구성됩니다.

카드 컴포넌트 가이드

  • 카드 컴포넌트는 card__content 안에 card__header와 card__main 영역으로 구조화됩니다.
  • card__header에는 card__status와 card__main이 포함됩니다.
  • card__status에는 상태 배지들이 badge-group badge-group--status로 그룹화됩니다.
  • card__main에는 타이틀, 카테고리, 요약글, 날짜, 태그가 포함됩니다.
  • card__categories에는 카테고리 배지들이 badge-group badge-group--category로 그룹화됩니다.
  • card__tags에는 태그들이 tag-list 컨테이너로 그룹화됩니다.
  • 필수 요소는 타이틀(card__title)과 요약글(card__summary)입니다. 카드 리스트에서는 제목이 <div> 태그로 구성되고, 각 카드는 <article> 태그로 감싸져 계층구조 문제를 방지합니다.
  • 요약글은 card__summary-label과 card__summary-text로 구성되며, 라벨 뒤에는 구분자(|)가 자동으로 추가됩니다.
  • 카드 링크 타입을 사용하려면 card--link 클래스를 추가하고 icon-link-b 아이콘을 사용합니다.
  • 버튼은 card__buttons 영역에 btn-container로 그룹화하여 배치합니다.
  • 하나의 마크업으로 PC와 모바일 환경을 모두 지원하며, CSS 미디어 쿼리로 반응형 레이아웃이 자동 적용됩니다.
  • 카드 리스트는 시스템 레이아웃을 사용하여 모바일에서는 1열에 1개, PC에서는 최대 1열에 3개까지 배치됩니다.
  • 카드 리스트의 모든 카드는 기본 카드와 동일한 구조를 가지며, 버튼 영역은 제외되어 정보 표시에 집중된 형태로 구성됩니다.
  • 모든 카드는 동일한 크기를 유지하며, 같은 행의 카드 높이는 자동으로 동일하게 조정됩니다.
