JavaScript 기능

셀렉트 컴포넌트의 동작을 위해 필요한 JavaScript 코드입니다. 접근성을 고려한 키보드 내비게이션과 스크린 리더 지원이 포함되어 있습니다.

참고: 아래 스크립트는 셀렉트 컴포넌트가 포함된 페이지에서 필요합니다. 기본 셀렉트의 경우 옵션 열기/닫기 동작과 키보드 접근성은 브라우저 기본 기능으로 제공되며, 다중 선택형의 경우 a11y.js 유틸리티의 initMultiSelect 함수를 통한 초기화가 필요합니다.
키보드 접근성:
  • Tab 키: 셀렉트 박스로 포커스 이동 (테두리 색상만 변경)
  • Enter / Space 키: 셀렉트 박스 활성화 및 옵션 표시 (화살표 아이콘 회전)
  • ESC 키: 열린 셀렉트 박스 닫기
<!-- 접근성 유틸리티 스크립트 추가 -->
<script src="../../assets/scripts/common/a11y.js"></script>

<script>
  window.addEventListener('DOMContentLoaded', function () {
    const A = window.App && App.a11y;

    // formMsg: 도움말 보이기/오류 숨기기/aria 연결
    if (A && A.formMsg) A.formMsg.init(document);

    // 멀티 셀렉트 향상 UI
    document.querySelectorAll('.select--multiple').forEach(ms => {
      const nativeSelect = ms.querySelector('.select__native');
      if (nativeSelect) {
        // JavaScript 활성화 시 네이티브 select 숨기기
        nativeSelect.style.display = 'none';
        nativeSelect.classList.add('visually-hidden');
        nativeSelect.tabIndex = -1;
        nativeSelect.setAttribute('aria-hidden','true');
      }
      A && A.initMultiSelect && A.initMultiSelect(ms);
    });   
  });
</script>

a11y.js의 주요 기능 (업데이트됨)

a11y.js 파일에는 셀렉트 컴포넌트의 접근성을 향상시키는 다양한 기능이 포함되어 있습니다:

  • 키보드 탐색: 화살표 키, Enter, Space, Escape 키 지원
  • 스크린 리더 지원: ARIA 속성 자동 관리 및 강제 리플로우를 통한 안정적인 재공지
  • 다중 선택 시 태그 관리: 태그 생성/삭제 및 키보드 접근성 자동 설정
  • 포커스 관리: ESC로 닫기 시 필드로 복귀, 포커스 아웃/클릭으로 닫기 시 복귀 안함
  • 네이티브 select 동기화: 커스텀 UI와 네이티브 select 자동 동기화
  • 성능 최적화: 윈도우 리사이즈 디바운싱(120ms), 개발 환경 디버깅 로그
  • 자동 announcer 생성: 스크린 리더 사용자에게 선택/해제 상태 자동 알림
