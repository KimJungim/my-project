Input 컴포넌트

기본 입력

단독으로 사용되는 기본 입력 필드로, placeholder와 disabled 상태를 지원합니다.

<div class="fd-input">
  <input id="f-basic" type="text" class="fd-input__el" placeholder="텍스트">
</div>

<div class="fd-input">
  <input id="f-basic" type="text" class="fd-input__el" value="텍스트" disabled>
</div>

전체 입력 디스플레이

라벨, 컨트롤, 메시지를 포함한 입력 필드의 전체 구조로, 폼 내에서 일관된 UI를 제공합니다.

*메시지 텍스트

<form class="fd-form">
  <div class="fd-field">
    <label for="f-text" class="fd-field__lbl">전체 영역 유형</label>
    <div class="fd-field__ctrl fd-field__ctrl--input">
      <div class="fd-input fd-input--full">
        <input id="f-text" type="text" class="fd-input__el" placeholder="텍스트" required>
      </div>
    </div>
  </div>

  <div class="fd-field">
    <label for="f-text" class="fd-field__lbl">라벨 텍스트</label>
    <div class="fd-field__ctrl fd-field__ctrl--input">
      <div class="fd-input fd-input--full">
        <input id="f-text" type="text" class="fd-input__el" placeholder="텍스트" required>
      </div>
    </div>
  </div>
  
  <div class="fd-field" data-help="helpID">
    <label for="f-text" class="fd-field__lbl">도움말 및 오류 메시지</label>
    <div class="fd-field__ctrl fd-field__ctrl--input">
      <div class="fd-input fd-input--full">
        <input id="f-text" type="text" class="fd-input__el" placeholder="텍스트" required>
      </div>
    </div>
    <!-- 도움말/오류 영역 -->
    <p id="helpID" class="fd-field__msg">*메시지 텍스트</p>
  </div>

  <div class="fd-field">
    <label for="f-text" class="fd-field__lbl">버튼 조합</label>
    <div class="fd-field__ctrl fd-field__ctrl--input fd-field__ctrl--with-button">
      <div class="fd-input fd-input--full">
        <input id="f-text" type="text" class="fd-input__el" placeholder="텍스트" required>
      </div>
      <button type="button" class="fd-field__btn btn btn--solid">버튼</button>
    </div>
  </div>
</form>

오류 상태

입력 오류 발생 시 시각적으로 강조되며, 오류 메시지를 함께 표시해 사용자의 인지를 돕습니다.

*메시지 텍스트

*올바른 값을 입력해주세요.

<form class="fd-form">
  <div class="fd-field fd-field--error" data-help="helpID" data-error="errorID">
    <label for="f-text-error" class="fd-field__lbl">라벨 텍스트</label>
    <div class="fd-field__ctrl fd-field__ctrl--input">
      <div class="fd-input fd-input--full">
        <input id="f-text-error" type="text" class="fd-input__el" placeholder="텍스트" required>
      </div>
    </div>
    <!-- 오류 메시지 -->
    <p id="helpID" class="fd-field__msg">*메시지 텍스트</p>
    <p id="errorID" class="fd-field__msg fd-field__msg--error" aria-live="polite">*올바른 값을 입력해주세요.</p>
  </div>
</form>
참고: fd-input--full 클래스는 입력 필드를 부모 컨테이너의 가용 너비 전체로 확장할 때 사용합니다. 그리드(col-span)와 함께 사용하면 반응형 레이아웃에서도 일관된 입력 영역 크기를 유지할 수 있습니다.
