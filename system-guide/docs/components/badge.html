배지 그룹 설명

배지 그룹은 여러 배지를 일정한 간격으로 배치할 때 사용합니다. 타입에 따라 다른 간격이 적용됩니다:

  • badge-group--status: 상태 타입 배지를 위한 그룹 (간격 4px)
  • badge-group--category: 카테고리 타입 배지를 위한 그룹 (간격 4px)
// 배지 그룹 SCSS 구조
.badge-group {
  display: inline-flex;
  align-items: center;
  
  &--status {
    gap: rem(4);  // 상태 타입 간격
  }
  
  &--category {
    gap: rem(4);  // 카테고리 타입 간격
  }
}
