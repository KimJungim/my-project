Button Group 컴포넌트

버튼 방향

버튼 그룹은 기본적으로 가로 배치로 정렬됩니다. 세로 배치로 정렬하고 싶을 때는 btn-group--column 클래스를 추가합니다.

기본형(가로 배치)

기본형(세로 배치)

<!-- 가로 배치 -->
<div class="btn-group">
  <button class="btn btn--solid">수정</button>
  <button class="btn btn--fill">삭제</button>
</div>
<!-- 세로 배치 -->
<div class="btn-group btn-group--column">
  <button class="btn btn--solid">수정</button>
  <button class="btn btn--fill">삭제</button>
</div>

버튼 정렬

버튼 그룹은 기본적으로 가로 배치로 정렬됩니다. 좌측, 센터, 우측으로 정렬하고 싶을 때는 btn-group--left, btn-group--center, btn-group--right 클래스를 추가합니다.

좌측 정렬

센터 정렬

우측 정렬

<!-- 좌측 정렬 -->
<div class="btn-group btn-group--left">
  <button class="btn btn--solid">수정</button>
  <button class="btn btn--fill">삭제</button>
</div>
<!-- 센터 정렬 -->
<div class="btn-group btn-group--center">
  <button class="btn btn--solid">수정</button>
  <button class="btn btn--fill">삭제</button>
</div>
<!-- 우측 정렬 -->
<div class="btn-group btn-group--right">
  <button class="btn btn--solid">수정</button>
  <button class="btn btn--fill">삭제</button>
</div>
