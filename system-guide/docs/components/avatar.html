활용 패턴

아래는 아바타 컴포넌트를 활용한 다양한 패턴입니다. 실제 구현과 코드 예시를 함께 제공합니다.

.avatar-info {
  display: inline-flex;
  align-items: center;
}

아바타 버튼형

<!-- 드롭다운이 필요한 경우 -->
<button class="avatar-btn" onclick="toggleDropdown(this)">
  <span class="avatar">
    <img class="avatar__image" src="..." alt="사용자 아바타">
  </span>
  <span class="avatar-btn__text">엘루오</span>
</button>

<!-- 드롭다운이 필요없는 경우 -->
<button class="avatar-btn">
  <span class="avatar avatar--text">
    <span class="avatar__text">엘</span>
  </span>
  <span class="avatar-btn__text">엘루오</span>
</button>
/* avatar-btn: 아바타와 텍스트를 함께 버튼으로 만들기 위한 클래스
 * - display: inline-flex로 설정
 * - 기본 화살표 아이콘 포함 (::after 가상요소)
 * - .is-active 또는 [aria-expanded="true"] 상태 관리
 */
.avatar-btn {
  display: inline-flex;
  align-items: center;
  /* 기타 스타일 */
}

JavaScript 기능

아래 스크립트는 버튼형 아바타에서만 사용되며, 화살표 아이콘의 모션을 추가하기 위한 것입니다. 버튼 클릭 시 화살표가 회전하는 애니메이션 효과를 제공합니다.

참고: 드롭다운 메뉴가 필요 없는 경우, onclick="toggleDropdown(this)" 속성과 아래 스크립트를 생략해도 됩니다. 스크립트는 아바타 버튼이 드롭다운 메뉴를 제어할 때만 필요합니다. 
<script>
  function toggleDropdown(btn) {
    // 클래스 토글 방식
    btn.classList.toggle('is-active');
    
    // ARIA 속성 토글 방식
    const isExpanded = btn.getAttribute('aria-expanded') === 'true';
    btn.setAttribute('aria-expanded', !isExpanded);
    
    // 관련 드롭다운 메뉴 표시/숨김 처리
    // ...
  }
</script>
